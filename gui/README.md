## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- npm
- python 3.7+

### Install Dependencies

Clone the repo

```sh
git clone <REPO-URL>
```

Install NPM Packages

```sh
cd client
npm install
```

### Run The Project

Client

```sh
cd client
npm start
```
